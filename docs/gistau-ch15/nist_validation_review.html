NIST / GISTAU Validation Review

Validation baseline page for thermodynamic kernels, backend tiers and low-temperature helium review.

Validation maturity

This phase introduces the first explicit NIST/GISTAU validation baseline for the cryogenic thermodynamic framework.

  • Fallback values remain approximations.
  • CoolProp is the first open-source executable candidate.
  • REFPROP remains canonical for gas-region engineering checks.
  • HEPAK remains priority below 5 K and for two-phase helium.

Known limitations

  • Fallback entropy is not publication-grade.
  • JT inversion behavior is not yet reproduced.
  • Two-phase dome reconstruction remains pending.
  • He-II transition behavior remains pending.
  • Exergy values are currently engineering approximations.
