## RGL: Recursive Governance Legacy

RGL replaces the previous MCP-like term for the Golden Oldies concept.

RGL means mature domains continue developing, but they increasingly act as seniority-weighted governance infrastructure.

Mature domains evolve from:

```text
capabilities
-> stabilizers
-> governance ancestors
-> recursive operational elders
-> convergence gravity infrastructure
```

## RGL seniority metrics

| Metric | Meaning |
|---|---|
| maturity | current stability |
| global_completion | ecosystem-level completion and trust |
| lineage_depth | number of dependent waves / domains |
| iterations | repeated reuse and refinement count |
| survivability_cycles | proven continuity under change |
| governance_authority | normalization power |
| seniority_rank | RGL-active, RGL-senior, or RGL-elder |
