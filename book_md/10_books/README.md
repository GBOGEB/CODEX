# 10_books canonical workspace

This directory groups the book, slide, and data binders for CODEX. Use the subfolders as follows:

- `src/`: Markdown book sources and manifests (e.g., `book_md/book_order.txt`).
- `20_slides/`: Slide engine, assets, and deck sources; outputs should render to `20_slides/output/` (gitignored).
- `30_data/`: Inputs/outputs used by books or slides; mirrors legacy `Input/` and `output/`.

See `CODEX_Index.yaml` for the indexed artefacts and build entry points.
